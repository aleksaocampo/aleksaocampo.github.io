NonoLang - The Programming Language

Introduction

NonoLang is a domain specific programming language meant to create Nonogram Puzzles.

FEATURES:

  • DESIGN: Takes user's custom nonogram design
  • CLUE: Creates row and column clues based on user's design
  • GENERATE: Generates a filled out version of the nonogram
  • BLANK: Creates a blank Nonogram of the user's design to be filled out
